## Projects

### `rag-knowledge-analyst/`
A Retrieval-Augmented Generation (RAG) pipeline that answers questions from uploaded documents.

- Gemini for answer synthesis with source citations
- Supports PDF, TXT, MD files

### `ai-safety-audit/`
An AI safety evaluation tool that tests LLM outputs for harmful content, bias, and jailbreak resistance.
- Red-teaming prompt library
- Severity scoring for safety failures
- Exportable audit report
